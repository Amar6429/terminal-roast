# Changelog

All notable changes to the **Terminal Roast** extension are documented here.

## 1.0.0
- **Initial Release**: The birth of the roast!
- Added terminal monitoring for non-zero exit codes.
- Integrated native system audio playback (Windows/Mac/Linux).
- Added Status Bar toggle for Muting/Unmuting.
- Included initial pack of meme sounds in the media folder.
